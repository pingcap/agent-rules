# Example inventory (pytidb/examples)

Use these as starting points for patterns and runnable demos.

- quickstart: end-to-end connect, create table, insert, vector search.
- basic: minimal CRUD and search.
- auto_embedding: server-side auto embedding across providers.
- custom_embedding: custom embedding function usage.
- vector_search: vector search app example.
- fulltext_search: fulltext search app example.
- hybrid_search: hybrid search app example.
- image_search: multi-modal image search app example.
- memory: agent memory demo (streamlit + CLI).
- rag: RAG demo with Ollama.
- text2sql: NL to SQL demo with Streamlit.
- vector-search-with-realtime-data: realtime vector search example.

Each example has its own README with setup steps and env vars.
